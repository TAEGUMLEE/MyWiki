# 마크다운
똘창 위키는 Wiki.js 엔진을 사용합니다.

Wiki.js는 마크다운(MarkDown) 문법을 사용합니다. 즉, 대부분의 마크다운 문법이 똘창 위키에 적용 됩니다.

이 페이지에서는 마크다운 문법에 대해서 간략하게 설명합니다.

# 헤더
헤더는 문단의 제목 입니다. 페이지의 섹션을 나누는 역할을 합니다.
헤더는 계층 구조를 가질 수 있습니다.

헤더는 해쉬(#)를 사용해서 생성합니다.

```markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6
```
